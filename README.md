# 🍽️ Calorie Lens: Food Photo → Calories & Health Insights

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://calorie-lens-count.streamlit.app/)

Calorie Lens is a **Streamlit app** that estimates **calories, macros, and health insights** from a single food photo using **Google Gemini Vision models**.  
It’s designed as a helpful demo and not medical advice.

👉 Try it live here: [https://calorie-lens-count.streamlit.app/](https://calorie-lens-count.streamlit.app/)

---

## ✨ Features
- 📸 **Image input** via file upload or camera
- 📊 **Automatic nutrition estimates** (calories, protein, carbs, fat)
- 📝 **Health assessment** with positives, watch-outs, and suggestions
- ⚡ **Adjustable portions** - scale estimates interactively
- 📂 **Export results** as JSON or CSV
- 🕑 **History panel** - revisit your last 5 analyses
- 🔑 Built with **Google Gemini Vision API**
